## ⚖️ 故障定性判据 (Diagnostic Criteria)

请根据工具返回的 JSON 数据中的统计指标（`start_val`, `mean_val`, `max_abs_val`）进行判断。

### 🟢 类型 A：安全故障 (Safe/Low Risk)
**定义**：故障虽触发但未导致主备星敏同时失效，姿态仍受控，风险可控  
**判定条件**：必须**同时满足**以下所有条件：

1.  **重启后加电成功**：
    *   参数：`星敏B加电状态（推断）` -> `start_val`
    *   规则：`start_val == 1`
2.  **遮挡状态短暂**：
    *   参数：`姿态系统遮挡诊断结果` -> `mean_val`
    *   规则：`mean_val < 0.5`
3.  **重启计数稳定**：
    *   参数：`星敏B故障重启计数` -> `max_abs_val`
    *   规则：`max_abs_val <= 1`

### 🔴 类型 B：危险故障 (High Risk)
**定义**：故障导致星敏无法恢复工作或引发姿态失稳风险  
**判定条件**：满足**任意一条**以下条件：

1.  **危险条件1名称**：`星敏B故障重启计数` 的 `max_abs_val > 1`，表明频繁重启（推断）
2.  **危险条件2名称**：`星敏B加电状态（推断）` 的 `start_val == 0` 且 `mean_val == 0`，表示未加电（推断）
3.  **危险条件3名称**：`姿态系统遮挡诊断结果` 的 `max_abs_val == 1` 且持续时间超过10分钟（推断）
